Rodapé da pagina

São inumeros os recursos, que já foram incluidos no Dreamweaver desde a sua primeira versão.

Com o Dreamweaver fica muito fácil a criação de páginas web responsivas e adaptáveis a qualquer tamanho de tela. Além de possuir um editor de várias linguagens inteligente, ainda é possivel ter a tela dividida em duas condições, visualizando o código e o design ao mesmo tempo, ou apenas uma das opções. Fora isso, ainda é possível fazer rapidas modificações utilziando apenas os recursos visuais.

Como trabalhamos ou fazemos.

Temos duas opções para usar o Adobe, podemos criar os conteúdos dos sites incorporando os elementos por itens nos menus, ou criarmos o site programado com código em HTML.

Explique o HTML.

O HTML é uma linguagem de progamação que defini a estrutura e o significado de uma página web, ele fornece base para outros recursos interativos e desenvolve portais e documentos eletrônicos on-line.

O que são as TAGs, descreva exemplos.

TAGs pode ter vários significados como uma palavra-chave, um trecho de códigos, aparelhos eletrônicos etc,

- Explique o CSS.

CSS é a sigla Cascading Style Sheets, que significa Folha de Estilo em Cascatas.

Explique UTF-8 e para que usamos.

UTF-8 é a codificação de caracteres mais comum da World Wide Web, e usamos para representar textos em vários idiomas da página web.
